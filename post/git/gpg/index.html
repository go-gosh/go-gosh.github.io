生成gpg密钥

1

gpg --gen-key

输出gpg密钥

1
2
3
4
5

# 输出公钥
gpg --armor --output public-key.txt --export [用户ID]

# 输出私钥
gpg --armor --output private-key.txt --export-secret-keys

输入gpg密钥

1

gpg --import [密钥文件]

Git使用

  • 上传之前导出的gpg公钥到github

  • git设置gpg密钥

1

git config --global user.signingkey {key_id}
  • 每次提交
1

git commit -S -m "..."
  • 可以设置全局提交
1

git config --global commit.gpgsign true

其他

在mac上使用的时候，提交遇到了以下问题：

1
2

error: gpg failed to sign the data
fatal: failed to write commit object

搜索了一下，居然大部分解决方式都是让关掉gpgsign。。。对此我感到汗颜。

直到有一篇博客提到了https://gist.github.com/repodevs/a18c7bb42b2ab293155aca889d447f1b

就是一个环境变量export GPG_TTY=$(tty)

而另一种方法是按照Jetbrains所言安装agent: https://www.jetbrains.com/help/idea/set-up-GPG-commit-signing.html
